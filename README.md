# clcltr

> This is a repository for testing npm package flow

## Installation
```
npm install clcltr --save
```

## Usage

```js
const calculator = new Clcltr()

calculator.add(1, 4) // 5

calculator.subtract(6, 4) // 2

calculator.multiply(2, 5) // 10

calculator.divide(12, 3) // 4

```
